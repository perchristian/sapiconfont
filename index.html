SAP-icon font in Axure prototypes

  1. Download the Axure stencil at: http://experience.sap.com/fiori-design/resources/downloads/
  2. Install the SAP-icon font on your computer.

Making SAP-icon available on Axure Share

For viewers of the prototype that does not have the font installed, you can add the web-font by adding a reference to the CSS-file: http://perchristian.github.io/sapiconfont/sapicons.css.

Here is how:

  1. Select "Publish to Axure Share" (F6)
  2. Click "Edit" configuration
  3. Select "Web Fonts"
  4. Click "+" to add new Web Font
  5. Enter name "SAP-icons" (or any other name)
  6. Paste this URL into the URL field: http://perchristian.github.io/sapiconfont/sapicons.css
  7. Click OK

The SAP-icon font used in this repository is currently v3.001
