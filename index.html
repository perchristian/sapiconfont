SAP-icon font in Axure prototypes

Download the Axure stencil at: http://experience.sap.com/fiori-design/resources/downloads/ Install the SAP-icon font on your computer.

Making SAP-icon available on Axure Share

For viewers of the prototype that does not have the font installed, you can add the web-font by adding a reference to the CSS-file: http://perchristian.github.io/sapiconfont/sapicons.css.

Here is how: 1 Select "Publish to Axure Share" (F6) 1 Click "Edit" configuration 1 Select "Web Fonts" 1 Click "+" to add new Web Font 1 Enter name "SAP-icons" (or any other name) 1 Paste this URL into the URL field: http://perchristian.github.io/sapiconfont/sapicons.css 1 Click OK

The SAP-icon font here is currently v3.001
